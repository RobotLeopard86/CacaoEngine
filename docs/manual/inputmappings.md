# Input Mappings

Below is a list of all valid key mappings in Cacao Engine.  
Any key using the `KP` infix means that it refers to the keypad section of the keyboard.  
The `SUPER` keys refer to what might be considered the Windows key (known as the Super key on Linux), or the Command key on macOS.
