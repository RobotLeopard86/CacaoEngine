class PerspectiveCamera : public Cacao::Camera

Perspective camera.

Public Functions

PerspectiveCamera(float fov = 60)

Create a perspective camera.

Warning

If the window isn’t open when this method is called, a division by zero error will occur

Parameters:

fov – The FOV in degrees
