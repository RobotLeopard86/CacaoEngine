class AssetManager

Manages the loading of assets and the asset cache.

Public Functions

std::future<AssetHandle<Shader>> LoadShader(std::string definitionPath)

Load a shader from a shader definition file See the page “Definition Files” in the manual for their format.

See also

Shader::Compile

Parameters:

definitionPath – The path to a definition file

Throws:

Exception – If the definition file does not exist, has the wrong format, or fails to compile

std::future<AssetHandle<Texture2D>> LoadTexture2D(std::string path)

Load a 2D texture from an image file.

See also

Texture2D::Compile

Parameters:

path – The path to an image

Throws:

Exception – If the image file does not exist or fails to compile

std::future<AssetHandle<Mesh>> LoadMesh(std::string location)

Load a mesh from a given model file.

The input format is as follows: (model file path):(mesh name)

See also

Model::Model,Mesh::Compile

Parameters:

location – The location to retrieve the mesh from

Throws:

Exception – If the location parameter has the wrong format, the model file does not exist, the model fails to load, the model does not contain the requested mesh, or the mesh fails to compile

std::future<AssetHandle<Sound>> LoadSound(std::string path)

Load a sound from a file.

Supports MP3, WAV, Ogg Vorbis, and Ogg Opus files

See also

Sound::Compile

Parameters:

path – The path to a sound file

Throws:

Exception – If the sound file does not exist, fails to be read, or fails to compile (which may happen if the audio system is uninitialized)

std::future<AssetHandle<Font>> LoadFont(std::string path)

Load a font face from a file.

Parameters:

path – The path to a font file

Throws:

Exception – If the font file does not exist or the font face cannot be loaded from it

inline void UncacheAsset(std::string assetID)

Remove an asset from the cache.

Note

For use by the engine only

Parameters:

assetID – The ID of the asset to remove from the cache

Throws:

Exception – If the specified asset ID does not exist in the cache

template<typename T>
inline AssetHandle<T> GetHandleFromPointer(T *ptr)

Get an AssetHandle from the asset’s stored pointer.

This is used for assets to be able to retrieve their own handle, typically for passing to another object

Note

For use by the engine only

Parameters:

ptr – A raw pointer to the asset (typically obtained from this)

Returns:

An asset handle, which will be a null handle if the asset was not found in the cache
