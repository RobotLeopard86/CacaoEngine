class Mesh : public Cacao::Asset

A mesh. Implementation is backend-dependent.

Public Functions

Mesh(std::vector<Vertex> vertices, std::vector<glm::uvec3> indices)

Create a mesh from a list of vertices and indices.

Note

Prefer to use AssetManager::LoadMesh over direct construction

Parameters:

  • vertices – The list of vertices

  • indices – The list of indices (each “index” is a triangle comprised of three positions in the vertex list)

inline ~Mesh() final

Delete the mesh and release compiled data if present.

void Draw()

Draw the mesh.

Note

For use by the engine only

Throws:

Exception – If not compiled or if not called on the engine thread

virtual std::shared_future<void> CompileAsync() override

Compile the raw mesh data into a format that can be drawn asynchronously.

Throws:

Exception – If the mesh was already compiled

Returns:

A future that will resolve when compilation is done

virtual void CompileSync() override

Compile the raw mesh data into a format that can be drawn synchronously.

Throws:

Exception – If the mesh was already compiled

Returns:

A future that will resolve when compilation is done

virtual void Release() override

Delete the compiled data.

Throws:

Exception – If the mesh was not compiled

inline virtual std::string GetType() const override

Gets the type of this asset. Needed for safe downcasting from Asset.
