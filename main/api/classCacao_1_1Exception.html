class Exception : public std::exception

A Cacao Engine exception. Do not subclass this. Prefer over standard library exceptions.

Note

Recommended to use CheckException to generate exceptions.

Public Functions

inline Exception(std::string description, unsigned int exceptionCode)

Create a new exception.

Parameters:

  • exceptionCode – The exception code to use. Must have been registered prior to use. See the page “Exception Codes” in the manual for more details.

  • description – The human-readable message to print if the condition is false

inline const char *what() const noexcept override

Get the exception message.

This method is inherited from std::exception.

Note

The exception message as a C string

  • definition – The meaning of the exception code
