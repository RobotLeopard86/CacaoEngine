# libcacaoaudiodecoder  
## Quick convienience library for decoding audio buffer-to-buffer

## About
libcacaoaudiodecoder is a simple library that decodes MP3, WAV, Ogg Vorbis, and Ogg Opus buffer-to-buffer.
