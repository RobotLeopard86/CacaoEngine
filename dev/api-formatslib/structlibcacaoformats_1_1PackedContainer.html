  • Defined in File libcacaoformats.hpp

struct PackedContainer

Loaded structure of a generic packed file format.

Note

When loading a container, use FromBuffer. When creating a container for exporting, use FromData.

Public Functions

std::vector<unsigned char> ExportToBuffer()

Export a PackedContainer to a buffer.

Throws:

std::runtime_error – If the container has no data or data compression fails

Returns:

A buffer representing the container, which can be written to a file whole

Public Members

const FormatCode format

Type of contents.

const uint16_t version

File type version.

const std::array<uint8_t, 64> hash

SHA-512 uncompressed payload hash.

const std::vector<unsigned char> payload

Decompressed payload data.

Public Static Functions

static PackedContainer FromBuffer(std::vector<unsigned char> buffer)

Create a PackedContainer from a loaded buffer.

Parameters:

buffer – A buffer containing the entire file contents of a packed file format

std::runtime_error – If the buffer is not a valid packed file or the hash in the buffer does not match the payload

Returns:

static PackedContainer FromData(FormatCode format, uint16_t ver, std::vector<unsigned char> data)

Create a PackedContainer from its contents.

Parameters:

  • format – The format code of the container

  • ver – The version of the format

  • data – The uncompressed data to be stored
