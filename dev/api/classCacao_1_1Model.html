  • Defined in File Model.hpp

Class Documentation

class Model

The representation of a model file.

Public Functions

Model(std::string filePath)

Load a model from a file path.

Note

Prefer to use AssetManager::LoadMesh if loading a mesh from a model file

Parameters:

filePath – The path to load from

Throws:

Exception – If the file does not exist, could not be parsed, or has no meshes

~Model()

Destroy the model (does not destroy meshes)

std::vector<std::string> ListMeshes()

List the meshes in this model.

Returns:

The mesh list

Mesh *ExtractMesh(std::string id)

Extract a mesh from this model.

Note

The extracted mesh, if modified, will have those modifications reflected in the model’s stored in-memory mesh. The mesh will not be deleted with the model.

Parameters:

id – The name of the mesh to extract

Throws:

Exception – If the model does not contain a mesh with the given name

bool HasMesh(std::string id)

Check if this model contains a given mesh.

Parameters:

id – The name of the mesh to search for
