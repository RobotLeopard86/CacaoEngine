Class Documentation

class AudioPlayer : public Cacao::Component

A component that plays audio.

Public Functions

AudioPlayer()

Create a new audio player.

Note

Generally, you shouldn’t call this constructor directly. Prefer creating this through Entity::MountComponent

Throws:

Exception – If the audio system is unitialized

~AudioPlayer()

Delete the audio player.

void Play()

Play the audio contained in sound.

Throws:

Exception – If the audio system is uninitialized, a audio is already playing, or sound is a null handle

void TogglePause()

Pause or unpase audio playback.

Throws:

Exception – If the audio system is uninitialized or no audio is playing

void Stop()

Stop audio playback.

Throws:

Exception – If the audio system is uninitialized or no audio is playing

bool IsPlaying()

Check if a audio is playing.

Throws:

Exception – If the audio system is uninitialized

Returns:

Whether a audio is playing or not

bool IsPaused()

Check if playback is paused.

Throws:

Exception – If the audio system is uninitialized

Returns:

Whether playback is paused or not

void SetLooping(bool val)

Set if the audio should loop.

Parameters:

val – If the audio should loop

Throws:

Exception – If the audio system is uninitialized

void SetGain(float val)

Set the gain value.

Parameters:

val – The new gain value

Throws:

Exception – If the audio system is uninitialized

void SetPitchMultiplier(float val)

Set the pitch multiplier.

Parameters:

val – The new pitch multiplier

  • 0-1: Deeper

  • 1: Normal

  • > 1: Higher

Throws:

Exception – If the audio system is uninitialized

void SetPlaybackTime(float timeInSeconds)

Set how far through playback the player is.

Parameters:

timeInSeconds – The new time in seconds

Throws:

Exception – If the audio system is uninitialized

bool GetLooping()

Check if the audio is looping.

Throws:

Exception – If the audio system is uninitialized

Returns:

Whether the audio is looping or not

float GetGain()

Get the current gain value.

Throws:

Exception – If the audio system is uninitialized

Returns:

The current gain value

float GetPitchMultiplier()

Get the current pitch multiplier.

Throws:

Exception – If the audio system is uninitialized

Returns:

The current pitch multiplier

float GetPlaybackTime()

Get the current playback time.

Throws:

Exception – If the audio system is uninitialized

Returns:

The current playback time in seconds
