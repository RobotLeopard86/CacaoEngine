static LogToken Engine(Level level = Level::Info)

Log a message from the engine.

Note

For use only by the engine internally.

Parameters:

level – The severity of the message (optional, defaults to Info)

struct LogToken

A small object used to make the logging API more ergonomic. Use operator<< with it just like with std::cout to write log messages (no ending newline or std::endl necessary)
