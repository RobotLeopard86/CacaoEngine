class UIView

Renderer for a Screen.

Public Functions

void Render()

Render the selected screen and swap buffers.

Throws:

Exception – If there is no screen to render

void Bind(int slot)

Attach the front buffer to a texture slot.

Note

For use by the engine only

Parameters:

slot – The texture slot to attach to

Throws:

Exception – If the view hasn’t been rendered, already bound, or if not called on the main thread

void Unbind()

Detach from the current slot.

Note

For use by the engine only

Throws:

Exception – If the view isn’t bound, or if not called on the main thread

inline void SetScreen(std::shared_ptr<Screen> s)

Set the current screen.

Parameters:

s – The new screen to render

inline std::shared_ptr<Screen> GetScreen()

Get the current screen.

Returns:

The current screen

inline void SetSize(glm::uvec2 sz)

Set the size of the rendered area.

Parameters:

sz – The new size in pixels

inline glm::uvec2 GetSize()

Get the size of the rendered area.

Returns:

The current size in pixels

inline bool HasBeenRendered()

Check if this view has been rendered.

Returns:

Whether the view has been rendered or not

UIView()

Create a new UI view.

Warning

If the rendering backend has not been initialized yet, this function will block until it is

Throws:

Exception – If the framebuffers could not be created

~UIView()

Destroy the UI view.
